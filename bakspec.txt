# Aroxima — Homepage Spec-Kit (Build-Ready)

## 0) Tech + Project Setup

* **Stack:** SvelteKit 2 / **Svelte 5**, TailwindCSS, TypeScript optional.
* **Folders**

  * `src/routes/+page.svelte` (homepage)
  * `src/lib/sections/*` (split sections)
  * `static/images/placeholders/*` (all temp images live here)
  * `src/lib/theme/tokens.css` (CSS variables)
* **Fonts:** Google Fonts — `Cinzel` (display, heroic serif) + `Inter` (body).
* **Tailwind Config:** enable container & animations; add safelist for glow utilities.
* **Components:** Use lots of components. Each major section should be inside its own component.

## 1) Theme Direction — “Rebirth of Science”

* **Not dark theme.** Soft daylight with warm, dusty undertone + biotech green glow accents.
* **Core palette (CSS vars in `tokens.css`):**

  ```css
  :root{
    --bg-base:#f4f2ec;          /* dusty parchment daylight */
    --bg-elev:#ffffffcc;        /* glass panels */
    --ink:#1c1f1b;              /* body text */
    --muted:#5e615a;            /* secondary text */
    --life:#4a935c;             /* revival green (primary) */
    --rust:#a86f3c;             /* oxidized accent */
    --sun:#f8e5b2;              /* hope light */
    --border:#e5dfcf;           /* soft border */
    --shadow:0 28px 60px -18px rgba(40,50,25,.25);
    --radius:18px;
    --blur:18px;
  }
  ```
* **Atmospherics:** subtle parallax **dust/fog/motes** layers; very slow drift; reduced-motion safe.
* **Layout tone:** **lots** of white space, long scroll, large typographic scale.

## 5) Atmospherics & Animation

### Background Layers (Hero + long page)

* `div[data-layer="fog"]` — subtle grain gradient, opacity .25, slow translateY
* `div[data-layer="motes"]` — tiny circles (CSS generated), slow floating keyframes
* Parallax: `transform: translateY(var(--scroll-factor))` via CSS sticky container

### Motion tokens (Tailwind `@layer utilities`)

```css
@keyframes drift { from{transform:translate3d(0,0,0)} to{transform:translate3d(4vmax,2vmax,0)}}
@keyframes pulse { 0%,100%{transform:scale(1)} 50%{transform:scale(1.02)}}
```

* Apply micro-pulse to CTA buttons and hero sunbeams (opacity gradient sweep)

### Reduced Motion

* If `prefers-reduced-motion: reduce`, disable all keyframes & parallax.
